# StrokeClaw 项目开源代码与组件使用情况说明

## 1. 结论
- 该项目**已使用开源代码与开源组件**，包括：
- Python 生态开源依赖（AI/医学影像/服务框架等）。
- 前端开源依赖（React/Vite 生态）。
- 构建产物中打包的开源运行时代码（React MIT 许可声明可见）。
- 源码中存在明确标注来源的开源实现借鉴/移植片段。
- 项目还集成了 MedGemma 模型资源，其许可为特定条款（非标准 MIT/Apache 文本）。

## 2. 检查范围与方法
- 扫描依赖清单：`pyproject.toml`、`uv.lock`、`requirements*.txt`、`frontend/package.json`、`frontend/package-lock.json`。
- 检索源码中的来源与许可线索：`license`、`github.com`、`ported from`、`borrowed` 等关键词。
- 复核构建产物中是否包含第三方开源代码许可头。

## 3. 已识别的开源组件与代码

证据：
- `pyproject.toml:7-30` 声明了主要依赖（节选）：

结论：

### 3.2 前端开源依赖（项目声明）
证据：
