# Viz LVO 风格界面部署指南

## 🎯 新界面特点

我已经创建了一个完全符合Viz LVO风格的专业医学影像界面：

### ✅ Viz LVO 风格特征
1. **纯黑背景** (#000000) - 无渐变、无装饰
2. **2x4网格布局** - 同时显示8个关键图像
3. **极简UI** - 无圆角、无阴影、无动画
4. **小字体** (10-13px) - 专业紧凑
5. **细线分隔** (1px) - 清晰但不突兀
6. **功能优先** - 每个元素都有明确用途
7. **专业提示** - 顶部显示AI免责声明

### ✅ 保留的所有功能
- 文件上传（CTA + NCCT）
- RGB合成和单通道显示
- AI灌注图（CBF、CBV、Tmax）
- 伪彩图生成和切换
- 脑卒中分析（偏侧选择、病灶分割、量化指标）
- 切片导航
- 数据下载
- 图像旋转90度

## 🚀 部署方法

### 方案A：直接替换（推荐）

```bash
# 1. 备份当前文件
copy templates\index.html templates\index_current.html

# 2. 使用Viz风格界面
copy templates\index_viz_style.html templates\index.html

# 3. 重启服务器
python app.py
```

### 方案B：并行测试

在 [`app.py`](app.py:1) 中添加新路由：

```python
@app.route('/viz')
def viz_style():
    return render_template('index_viz_style.html')
```

然后访问：
- 原界面: `http://localhost:5000/`
- Viz风格: `http://localhost:5000/viz`

## 🔧 使用说明

### 1. 上传文件
1. 点击"CTA文件"选择按钮
2. 点击"NCCT文件"选择按钮
3. 点击"开始处理"
4. 等待AI推理完成

### 2. 查看图像
- **网格显示**: 8个图像同时显示在2x4网格中
- **切片导航**: 使用底部滑块或◀▶按钮
- **状态指示**: 每个cell右上角显示✓表示已加载

### 3. 伪彩图功能
- **方法1**: 点击顶部"伪彩图"按钮，为当前切片生成所有伪彩图
- **方法2**: 点击CBF/CBV/Tmax cell右下角的"伪彩"按钮，切换单个图像的伪彩图模式

### 4. 脑卒中分析
1. 点击顶部"脑卒中分析"按钮，打开右侧面板
2. 选择偏侧（右脑/左脑/双侧）
3. 点击"开始分析"
4. 查看结果：
   - 病灶可视化（半暗带、核心梗死、综合显示）
   - 量化指标（体积、不匹配比例）
   - 底部栏实时显示关键指标

### 5. 数据下载
- 点击顶部"下载"按钮
- 自动下载当前切片的所有数据（NPY、CBF、CBV、Tmax）
