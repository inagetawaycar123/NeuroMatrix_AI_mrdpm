### TipTap 编辑器功能

```
工具栏布局:
┌────────────────────────────────────────────────────────┐
│ [B] [I] [S] │ [H2] [•] [1.] │ [🔗] [🖼️] │ [↶] [↷]     │
├─ 文本 ─────┼─ 结构 ─────────┼─ 媒体 ────┼─ 历史 ─────┤
│ 加粗 Ctrl+B │ 标题 Ctrl+2 │ 链接   │ 撤销 Ctrl+Z │
│ 斜体 Ctrl+I │ 列表 Ctrl+8 │ 图片   │ 重做 Shift+Z│
│ 删除 Ctrl+X │ 编号 Ctrl+7 │        │            │
└────────────────────────────────────────────────────────┘

编辑内容示例:
──────────────────────────────────────────
核心梗死区位于左侧大脑半球，体积约 25.3 ml。
病灶界限清晰，灌注信号明显降低。

灌注参数分析：
• CBF (脑血流量)：核心梗死区显著降低
• CBV (脑血容量)：与梗死灶分布相符
• Tmax (平均通过时间)：延迟区域远大于核心区
──────────────────────────────────────────
```

## 🔗 访问报告系统

```
http://localhost:5000/report/1?file_id=test123
              ↓
      Flask 路由处理
              ↓
      检查 static/dist/index.html
              ↓
      返回编译后的 React 应用
              ↓
      React 应用初始化
         ├─ 解析 URL 参数 (patientId=1)
         ├─ 读取 sessionStorage (analysis_data)
         └─ 获取 API: /api/get_patient/1
              ↓
      显示报告 (查看模式)
              ↓
      点击"编辑报告" → 编辑模式
              ↓
      使用 TipTap 编辑内容
              ↓
      点击"保存报告" → POST /api/save_report
              ↓
      保存到 Supabase
```
