# 📸 医学影像上传使用指南

## ✅ 现在已修复！图像功能已优化

系统已更新为 **混合模式** 处理医学影像：

```
用户上传医学影像 → 系统验证完整性 → 提示用户补充影像描述
                                    ↓
                           用户在问题中提供具体发现
                                    ↓
                        AI基于发现进行准确分析
```

---

## 🎯 正确的使用方式

### 步骤 1️⃣: 上传医学影像

在聊天界面点击 **📷 图像按钮**：
- 选择 JPG 或 PNG 格式的医学影像
- 可上传 CT、MRI、X光、超声等任何医学影像
- 支持一次上传多个图像

### 步骤 2️⃣: 在问题中描述影像发现

**❌ 错误的方式**：
```
"这是一张CT，是否可以溶栓？"
"患者影像显示异常，怎么处理？"
"请分析这个医学影像。"
```

**✅ 正确的方式**：
```
"患者CT头部显示左侧MCA供血区低密度灶，
面积约2cm×3cm，周围无明显水肿。
NIHSS 14分，发病2小时。
请问是否适合IV-tPA溶栓？"
```

### 步骤 3️⃣: 获得准确的医学建议

系统会根据您提供的：
- ✓ 具体的影像学表现
- ✓ 患者临床数据（NIHSS、年龄等）
- ✓ AHA/ASA 指南

生成精准的医学建议！

## 📝 实际案例

### 案例 1: 急性缺血性卒中评估

**输入**:
```
【上传】: 患者头部CT图像

【问题】: 
患者CT头部显示左侧MCA起始段血栓阴影，
相应供血区见点片状低密度灶，面积约3cm×2cm。
患者68岁女性，NIHSS 12分，
发病时间2.5小时，
血压160/95 mmHg，
血糖6.2 mmol/L。

请问：
1) 该患者是否适合IV-tPA溶栓？
2) 是否存在溶栓禁忌症？
3) 建议的时间窗是什么？
```

**输出**:
```
✅ 根据患者CT和临床数据的综合分析：

【卒中类型】: 缺血性卒中（ischemic stroke）
【影像诊断】: 左侧MCA供血区急性缺血灶

【IV-tPA 溶栓评估】:
✓ 年龄: 68岁 - 在推荐范围内（IIA级）
✓ 时间窗: 2.5小时 - 在4.5小时窗内（I级）
✓ NIHSS: 12分 - 中度卒中，符合溶栓指征
✓ 影像: 无出血证据
✓ 血压: 160/95 - 可接受范围

【建议】:
根据AHA/ASA指南，该患者适合进行IV-tPA静脉溶栓治疗
证据等级: I级 (强烈推荐)

【需要注意】:
1. 血压>185/110需先控制到安全范围
2. 排除其他溶栓禁忌症
3. 需要签署知情同意书
4. 溶栓后需密切监测神经体征变化

【医学免责声明】...
```

### 案例 2: 医学影像异常检查

**输入**:
```
【上传】: 患者MRI脑部图像

【问题】:
患者MRI脑部检查：
• DWI序列: 右侧内囊后肢急性高信号
• T2 FLAIR: 右侧内囊区异常信号
• MRA: 右侧MCA中段见血栓信号

患者突然出现：
• 左侧肢体无力（突然发作）
• 言语困难
• 发病时间: 1.5小时

请问: 患者是否应该立即进行溶栓或取栓？
```

**输出**:
```
✅ 基于MRI发现的紧急评估：

【影像诊断】: 
• 急性缺血性卒中（内囊后肢供血区）
• MCA中段血栓存在

【临床紧急程度】: 🚨 高度紧急

【治疗建议】:
1. **IVT (IV-tPA)**: 符合条件且发病在时间窗内
   建议立即进行IV-tPA溶栓 (I级推荐)
   
2. **EVT (血管内治疗/取栓)**:
   MCA中段血栓+ MRI确认急性卒中
   应评估是否适合机械取栓 (IIA级推荐)

【立即行动项】:
□ 通知神经外科/介入放射科
□ 准备血管造影
□ 评估全身状况和禁忌症
□ 快速获取患者知情同意

【医学免责声明】...
```

## 🎓 医学术语参考

### 常见放射学发现：

| 术语 | 含义 | 在卒中中的意义 |
|------|------|---------------|
| 低密度灶 | 较周围组织密度低 | 常见于缺血性卒中 |
| 高信号 | MRI上异常增强 | DWI上提示急性缺血 |
| 血栓阴影 | CT/MRA上血管内异物 | 血管堵塞 |
| 水肿 | 脑组织肿胀 | 卒中后24-48h出现 |
| MCA | 大脑中动脉 | 卒中最常见部位 |
| 供血区 | 某血管供血的脑区域 | 血管堵塞时相应区域缺血 |

### 常见缩写：

- **CT**: 计算机断层扫描
- **MRI**: 磁共振成像
- **DWI**: 弥散加权成像（对急性缺血最敏感）
- **MCA**: 大脑中动脉
- **ACA**: 前大脑动脉
- **PCA**: 后大脑动脉
- **NIHSS**: 美国国家卫生研究院卒中量表

## ⚠️ 重要提示

### 系统的能力和限制

✅ **系统能做的**:
- 基于您提供的临床数据进行分析
- 参考 AHA/ASA 指南给出建议
- 标注证据等级
- 提供医学参考

❌ **系统不能做的**:
- 自动识别和诊断医学影像（需要医生补充发现）
- 替代专业放射科医生的诊断
- 进行远程诊断
- 替代医生的临床判断

### 使用责任

1. **医学专业人士用途**: 仅供临床参考和辅助决策
2. **需要专业医生**: 所有诊断和治疗决策由持证医生做出
3. **知情同意**: 所有治疗方案需获得患者同意
4. **持续监测**: 治疗效果需要定期评估和调整

## 🔧 故障排除

### 问题 1: 上传图像后，系统说"无法分析"

**解决**:
- ✓ 在问题中补充具体的影像学发现
- ✓ 例如: "CT显示左侧MCA血栓，大小约5mm"
- ✓ 系统会基于您的描述进行分析

### 问题 2: 获得的建议太笼统

**解决**:
- ✓ 提供更多临床细节
- ✓ 补充患者的既往史和用药信息
- ✓ 描述具体症状而不是笼统的"不适"

### 问题 3: 系统无法识别特殊医学术语

**解决**:
- ✓ 使用常见的医学术语
- ✓ 提供通俗解释（例如: "脑梗塞" = 缺血性卒中）
- ✓ 查看上方的"医学术语参考"
